# Настройка Telegram для получения заявок

## Переменные окружения для Netlify

Добавьте следующие переменные окружения в настройках вашего проекта на Netlify:

### 1. Откройте настройки Netlify
1. Перейдите на https://app.netlify.com
2. Выберите ваш проект `autoace24`
3. Перейдите в **Site settings** → **Environment variables**

### 2. Добавьте переменные окружения

Нажмите **Add a variable** и добавьте следующие переменные:

#### TELEGRAM_BOT_TOKEN
```

```

#### TELEGRAM_CHAT_ID_1
```
7552690430
```

#### TELEGRAM_CHAT_ID_2
```
6947421295
```

### 3. Сохраните изменения

После добавления всех трех переменных:
1. Нажмите **Save**
2. Netlify автоматически передеплоит сайт

### 4. Проверьте работу

1. Откройте ваш сайт https://autoace24.netlify.app
2. Заполните и отправьте тестовую заявку
3. Проверьте, что сообщение пришло обоим пользователям в Telegram

## Важно!

- Заявки будут приходить **обоим** пользователям одновременно
- Убедитесь, что бот запущен (напишите боту `/start`)
- Если сообщения не приходят, проверьте правильность токена и chat ID

## Как получить свой Chat ID

Если нужно добавить еще пользователей:
1. Напишите боту @userinfobot команду `/start`
2. Бот пришлет ваш Chat ID
3. Добавьте новую переменную окружения в Netlify
